# Additional Managed Services

Welkin recommends the following Operators or Helm Charts:

- [Argo CD Helm Chart](https://argoproj.github.io/argo-helm/)
- [Jaeger operator](https://github.com/jaegertracing/jaeger-operator)
- [Zalando operator for PostgreSQL](https://github.com/zalando/postgres-operator)
- [RabbitMQ Cluster Operator for Kubernetes](https://www.rabbitmq.com/kubernetes/operator/operator-overview.html)
- [Spotahome operator for Redis](https://github.com/spotahome/redis-operator)
